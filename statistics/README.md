# Estadística y Probabilidad

Base matemática de todo ML. Se pregunta directamente en entrevistas de ciencia de datos.

| Archivo | Contenido |
|---|---|
| `descriptive.py` | Media, mediana, moda, varianza, std, percentiles |
| `distributions.py` | Normal, Binomial, Poisson — implementadas desde cero |
| `hypothesis_testing.py` | t-test, chi-square, p-value, intervalos de confianza |
| `probability.py` | Probabilidad condicional, Bayes, combinatoria |

**Conceptos clave que se preguntan en entrevistas:**
- ¿Cuándo usar media vs mediana?
- ¿Qué es el p-value y cómo se interpreta?
- ¿Qué es el Teorema de Bayes?
- ¿Diferencia entre distribución normal y binomial?
