# Customer Segmentation — Clustering

**Dataset:** Mall Customers  
**Tipo:** Clustering no supervisado  
**Objetivo:** Segmentar clientes por comportamiento de compra

## Flujo

1. `eda.ipynb` — análisis exploratorio, distribuciones de edad/ingreso/gasto
2. `model.py` — K-Means desde cero + elbow method
3. `README.md` — este archivo

## Resultado esperado

Segmentos de clientes con perfiles identificables (ej. alto ingreso/bajo gasto)
