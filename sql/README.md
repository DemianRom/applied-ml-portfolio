# SQL para análisis de datos

SQL es indispensable en cualquier rol de datos. Aquí están los patrones más frecuentes.

| Archivo | Contenido |
|---|---|
| `basics.sql` | SELECT, WHERE, JOIN, GROUP BY, ORDER BY |
| `window_functions.sql` | ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD, SUM OVER |
| `aggregations.sql` | HAVING, subqueries, CTEs, UNION |
| `interview_problems/` | Problemas típicos de entrevista resueltos |

**Patrones que más se preguntan:**
- Second highest salary
- Running totals con SUM OVER
- N-th record con ROW_NUMBER
- Self joins
- Detectar duplicados
