# NumPy y Pandas

Herramientas base para análisis de datos. Se dan por sentado en entrevistas — 
pero estas implementaciones sirven como referencia y repaso rápido.

| Archivo | Contenido |
|---|---|
| `numpy_basics.py` | Arrays, broadcasting, operaciones matriciales, álgebra lineal |
| `pandas_basics.py` | DataFrames, groupby, merge, pivot, apply |
| `data_cleaning.py` | Nulls, outliers, encoding, normalización |
