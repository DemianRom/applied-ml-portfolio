# Exploratory Data Analysis (EDA)

Lo primero que haces en cualquier proyecto real. Se pregunta en entrevistas de datos.

| Archivo | Contenido |
|---|---|
| `univariate_analysis.py` | Distribuciones, histogramas, boxplots, outliers |
| `bivariate_analysis.py` | Correlaciones, scatter plots, heatmaps |
| `feature_engineering.py` | Creación de features, transformaciones, encoding |

**Preguntas frecuentes en entrevistas:**
- ¿Cómo detectas y tratas outliers?
- ¿Qué haces con valores nulos?
- ¿Cómo manejas features categóricas?
