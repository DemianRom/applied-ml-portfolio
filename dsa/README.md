# DSA esencial para roles de datos

En roles de ML/datos no se pregunta Blind75 completo, pero sí los fundamentos.
Estas son las estructuras y algoritmos que aparecen con más frecuencia.

| Archivo | Tema |
|---|---|
| `arrays_hashing.py` | Arrays, hashmaps, frecuencias |
| `sorting.py` | Merge sort, quick sort, cuándo usar cada uno |
| `binary_search.py` | Búsqueda binaria y variantes |
| `recursion.py` | Recursión base para árboles y DP |
