# Big-O Cheatsheet

## Complejidades más comunes

| Notación | Nombre | Ejemplo |
|---|---|---|
| O(1) | Constante | Acceso a array por índice |
| O(log n) | Logarítmica | Binary search |
| O(n) | Lineal | Recorrer un array |
| O(n log n) | Linealítmica | Merge sort |
| O(n²) | Cuadrática | Bubble sort |
| O(2ⁿ) | Exponencial | Fibonacci recursivo |
| O(n!) | Factorial | Permutaciones |
